Mobil Kit Remote Control

Mobil KRC adalah mainan alternatif yang dibuat khusus untuk anak SD untuk mengurangi dampak game online. Mobil ini diciptakan dalam rangka mengikuti ajang PKM KC 2022. Proyek tersebut berhasil mendapatkan pendanaan dari Kementerian Pendidikan, Kebudayaan, Riset, dan Teknologi (Kemendikbudristek).

  • mengembangkan aplikasi mobile dengan Kodular untuk fungsi remote control.
  • mengintegrasikan koneksi WiFi untuk komunikasi antar perangkat.
  • Mengimplementasikan kontrol servo untuk fungsi gripper.
  • menyediakan fitur monitoring persentase baterai.

BPSDMP

Aplikasi BPSDMP dibangun saat saya mengikuti sertifikasi kompetensi dari VSGA (Vocational School Graduate Academy), yang merupakan program Digitalent X Kominfo. Saya mengembangkan aplikasi ini dengan PHP Native dan Boostrap 5.

  • Implementasi CRUD pada kegiatan.
  • Fitur login dengan pengelolaan sesi.
  • Menampilkan kegiatan terkini berdasarkan waktu di halaman beranda.
  • Menggunakan paginasi pada halaman kegiatan untuk mempermudah navigasi.

Web Forecast

Aplikasi ini memprediksi jumlah siswa baru di SMK Gajah Mada Banyuwangi dengan menggunakan metode Regresi Linier untuk memodelkan hubungan antara variabel dependen dan independen. Website ini bermanfaat untuk mempertahankan jurusan diminati dan mengganti yang kurang diminati. Saya mengembangkannya dengan menggunakan framework Laravel.

  • CRUD data siswa.
  • Penggunaan grafik untuk visualisasi.
  • Prediksi jumlah siswa baru secara umum.
  • Prediksi jumlah siswa baru per jurusan.
